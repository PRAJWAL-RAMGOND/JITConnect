# 🚀 JITConnect Quick Start Guide

## Welcome to JITConnect!

Your all-in-one college social network platform with AI-powered features.

---

## 🎯 Getting Started

### 1. First Time Setup
1. Open `login.html` in your browser
2. Click "SIGN UP" to create an account
3. Fill in your details (use @jit.ac.in email)
4. Select your role (Student/Faculty/Alumni)
5. Complete role-specific information
6. Click "CREATE ACCOUNT"

### 2. Login
1. Enter your email and password
2. Click "LOGIN"
3. You'll be redirected to your dashboard

---

## ⚡ Quick Actions

### Keyboard Shortcuts
- `Ctrl/Cmd + K` - Focus search bar
- `Ctrl/Cmd + N` - Create new post
- `Escape` - Close any modal/popup

### Dashboard
- **Create Post**: Type in the text area, select category, click POST
- **Like Posts**: Click the heart icon (animated particles!)
- **Comment**: Click comment button, type, press Enter
- **Share**: Click share button to copy link

### Jobs Board
- **Search Jobs**: Use the search bar at top
- **Filter**: Click tabs (All/Internships/Full-Time/On-Campus)
- **Save Job**: Click star icon on job card
- **Apply**: Click "Apply Now" button
- **Compare**: Click compare icon, select up to 3 jobs

### Alumni Matcher Chatbot
- **Quick Access**: Click purple bot icon (bottom-right)
- **Voice Input**: Click microphone icon to speak
- **Full Matcher**: Click "Open Full Matcher" for detailed matching
- **Suggested Questions**: Use pre-written questions for quick help

## 🔧 Troubleshooting

### Common Issues

**Can't Login?**
- Check email format (@jit.ac.in)
- Verify password (min 8 characters)
- Clear browser cache
- Try different browser

**Page Loading Slow?**
- Check internet connection
- Clear browser cache
- Disable browser extensions
- Use latest browser version

**Chatbot Not Working?**
- Ensure you're logged in as student
- Refresh the page
- Check browser console for errors
- Try different browser

**Images Not Loading?**
- Check internet connection
- Wait for lazy loading
- Refresh the page
- Clear browser cache
